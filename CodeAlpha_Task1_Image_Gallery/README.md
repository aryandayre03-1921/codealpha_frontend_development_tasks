# 🖼️ 3D Rotating Image Gallery

A modern and interactive **3D Rotating Image Gallery** built using **HTML, CSS, and JavaScript**. This project showcases stunning image collections in a continuously rotating 3D carousel with category-based navigation, hover effects, reflections, and responsive design.

---

# ✨ Features

## 🎨 Modern User Interface

* Attractive Gradient Background
* Professional Gallery Layout
* Smooth User Experience
* Responsive Design
* Clean Navigation Menu

# 💻 Installation Guide

## Step 1

Clone Repository

```bash
git clone https://github.com/your-username/3d-rotating-image-gallery.git
```

## Step 2

Open Project Folder

## Step 3

Ensure all images are placed in their correct folders:

```text
images/
│
├── nature/
├── car/
├── birds/
├── wild amimal/
├── flower/
├── bike/
└── beach/
```

## Step 4

Run:

```text
index.html
```
