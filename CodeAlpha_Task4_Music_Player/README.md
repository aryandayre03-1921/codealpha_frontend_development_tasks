# 🎵 Neon Music Player

A modern and visually stunning Music Player built using **HTML, CSS, and JavaScript**. This project features an animated neon interface, colorful glowing effects, rotating music disc animation, playlist management, volume controls, progress tracking, and responsive design.

# ⚠️ Important Notes

1. All music files must be inside the `music` folder.
2. File names must exactly match the names used in `script.js`.
3. Incorrect file names will trigger an error alert.
4. Audio format should be `.mp3`.
