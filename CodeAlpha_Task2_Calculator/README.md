# 🧮 Neumorphic Calculator

A modern and visually attractive calculator built using **HTML, CSS, and JavaScript** with a beautiful **Neumorphic UI Design**, animated gradient background, glowing button effects, and smooth interactions.

---

## ✨ Features

### 🎨 User Interface

* Modern Neumorphic Design
* Soft shadow effects for realistic depth
* Responsive layout
* Clean and elegant display screen
* Smooth animations and transitions

### 🌈 Visual Effects

* Animated multi-color gradient background
* Rainbow glowing hover effects
* Interactive button press animation
* Dynamic color-changing button effects
* Neon-style glow effects

### 🧮 Calculator Functions

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Percentage (%)
* Decimal calculations
* Positive/Negative toggle (+/-)
* Clear screen (AC)
* Instant result calculation (=)

### ⚡ User Experience

* Smooth hover animations
* Pressed button effect
* Fast calculations
* Error handling for invalid expressions
* Simple and intuitive interface

## 📋 How It Works

### 1. HTML

* Creates calculator layout
* Defines display screen
* Adds calculator buttons
* Connects CSS and JavaScript files

### 2. CSS

* Designs Neumorphic UI
* Creates animated gradient background
* Adds glow effects
* Implements hover animations
* Handles responsive styling

### 3. JavaScript

* Handles button clicks
* Updates display values
* Performs calculations
* Toggles positive/negative values
* Clears display
* Shows error messages for invalid inputs
